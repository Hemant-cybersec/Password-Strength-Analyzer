# Password Strength Analyzer

## Overview
The Password Strength Analyzer is a web-based cybersecurity project developed using HTML, CSS, and JavaScript. It helps users analyze password strength and provides security recommendations for creating stronger passwords.

## Features
- Password Strength Analysis
- Security Score (0-100)
- Password Requirements
- Show/Hide Password
- Copy Password
- Progress Bar
- Security Recommendations
- Feedback Form
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download the project.
2. Open index.html in your browser.
3. Enter a password.
4. Click "Analyze Password".
